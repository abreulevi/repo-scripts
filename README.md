# 🛠️ Scripts e Toolkits de Automação

Uma coleção centralizada de scripts, templates e ferramentas de linha de comando voltadas para infraestrutura, auditoria de sistemas, troubleshooting e hardening.

## 📖 Propósito do Repositório

Este repositório é um catálogo em constante evolução. O objetivo aqui é manter um repositório organizado de soluções práticas para o dia a dia de administração de sistemas e segurança da informação.

Aqui você encontrará desde templates de código para padronizar novas criações até ferramentas independentes escritas em diferentes linguagens (PowerShell, Bash, etc.), focadas em resolver problemas específicos de auditoria, levantando informações para análise e hardening. Também para configuração e análise de ambientes.

## 📂 Estrutura do Repositório

O repositório está organizado de forma modular, baseada na finalidade das ferramentas ou no ambiente alvo. Novas categorias serão adicionadas conforme a necessidade:

* 📋 **[Templates](./Templates/)**: Códigos base e esqueletos estruturais (como motores de menu interativo) desenvolvidos para acelerar a criação de novas ferramentas de forma padronizada.
* 🌐 **[Análise de Conexão de Redes](./Powershell/analise_rede.ps1)**: Automações focadas em testes de conectividade, mapeamento de rede e análise de portas/tráfego.
* 💻 **[Auditoria e Hardening](./Powershell/auditoria_hardening.ps1)**: Scripts voltados para o levantamento de informações locais do sistema operacional, análise de configurações e vetores de privilégio.
* 🐧 **[Linux-BashTools](./LinuxBash/)**: *(Em Breve)* Coleção de ferramentas e auditorias utilizando scripts em Bash para ambientes Linux.

## 🚀 Como Utilizar

Como este repositório reúne automações independentes, não existe um fluxo único de configuração. Consulte a documentação de cada script para instruções específicas.

## ⚠️ Aviso Legal

Os scripts contidos neste repositório foram desenvolvidos para fins de estudo, administração de ambientes próprios e auditorias devidamente autorizadas. Não execute estas ferramentas em redes ou sistemas para os quais você não possui permissão explícita de atuação.
